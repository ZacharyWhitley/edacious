implementation of the withings api in java edacious: Having an insatiable appetite; voracious, ravenous, piggish. This is a complete implementation of the withings api in java. Releases of Edacious map to the svn commit number. This makes management of the project easier. I've spent a bunch of hours working on this library so far. It is open source with an Apache License 2.0 because I believe that libraries like this should be available for everyone to use.
